# Alumol Lifetime Cost Calculator

This folder is ready for GitHub Pages.

Files:

- `index.html` - front page linking to both calculator versions
- `website-version.html` - public-facing website version
- `account-management-version.html` - internal account management version with manual Alumol quoted pricing

To publish:

1. Create or open a GitHub repository.
2. Upload these files to the repository root.
3. In GitHub, go to Settings > Pages.
4. Set the source to the main branch and root folder.
5. Save, then use the GitHub Pages URL once it is generated.

Note: if GitHub Pages is public, both calculator versions are public to anyone with the link. Use a private repository or internal sharing controls if the account management version should remain internal.
